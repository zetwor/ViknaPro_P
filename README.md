# Технологічна практика 
## Виконав стдент 44 групи - Яковлев Ілля

## Опис проєкту
Під час технологічної практики розроблено вебсайт-візитку для компанії, що спеціалізується на монтажі металопластикових вікон.  
Сайт виконує роль онлайн-представництва компанії та надає клієнтам інформацію про послуги, етапи виконання робіт і контакти.

## Посилання на сайт - https://zetwor.github.io/ViknaPro_P/

## Основні функції
- Головна сторінка з інформацією про компанію  
- Розділ **«Послуги»** (монтаж, демонтаж, підвіконня, відливи)  
- Розділ **«Етапи робіт»** з покроковим описом  
- Сторінка **«Контакти»** з формою зворотного зв’язку  

## Технологічний стек
- **Frontend:** HTML, CSS, JavaScript   
- **Методологія:** Mobile First
- **Інструменти:** Visual Studio Code, GitHub  
- **Дизайн:** Власний макет (draw.io)
